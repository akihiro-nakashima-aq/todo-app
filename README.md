# Todo App

Next.js + TypeScript + Tailwind CSS + SQLite で作ったミニマルなTodoアプリ。

## 技術スタック

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **SQLite** (better-sqlite3)
- **Server Actions**

## 機能

- Todo追加・削除
- 完了/未完了のトグル
- フィルター表示（全件 / 未完了 / 完了）

## 開発サーバーの起動

```bash
npm run dev
```

[http://localhost:3000](http://localhost:3000) をブラウザで開く。

## ビルド

```bash
npm run build
npm start
```
